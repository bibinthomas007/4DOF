
This package contains the files to make a 4dof robotic arm which can autonomously detect packages, pick it and place it in a  drop zone

# Hardwares used
1) MG995  
Stall Torque @6.6v : 12 kg-com
current load: 1200mA

2) 1501 MG
Stall Torque @6v : 17 kg-com
current load: 3A

3) OT533OM    
Stall Torque @7.4v : 35.5 kg-com
current load: 3.8A


Servo pulse width is 500 to 2500 for both orange 35 kg-cm servo and 25kg-cm servo
